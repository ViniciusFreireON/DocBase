# Melhorias e Correções – Slides Revisão HTML/CSS

## Sobre o documento

O arquivo **slides-revisao-html-css-corrigido.md** contém 16 tópicos sobre estrutura HTML essencial (DOCTYPE, head, body), SEO no `<head>`, semântica e hierarquia de títulos, acessibilidade (alt, label), ARIA e tabindex, tipos de imagem (PNG, JPEG, WebP, SVG), favicons, introdução ao CSS (inline, interno, externo), regra CSS, seletores, cor e tipografia, box model e comentários/boas práticas de CSS.

---

## Correções aplicadas

### Parágrafo deslocado
| Local | Problema | Correção |
|-------|----------|----------|
| Slide "Comentários e Boas Práticas de CSS" | Texto sobre "checagem de caminhos relativos", "index.html (raiz)", "pages/contato.html" e "assets/..." pertencia ao slide de Estrutura de Projeto / multipage | Substituído por descrição adequada: comentários CSS (`/* */`) e boas práticas (nomes descritivos, indentação, CSS externo) |

### Atributo lang
| Local | Original | Correção |
|-------|----------|----------|
| Recomendação de idioma | `lang="pt-BR"` | **lang="pt-br"** (minúsculas, em conformidade com BCP 47 e boas práticas HTML5) |

### Organização
- Conteúdo reunido em 16 tópicos na ordem lógica: estrutura HTML → SEO → semântica → acessibilidade → ARIA → imagens → favicons → CSS (inline/interno/externo) → regra CSS → seletores → cor e tipografia → box model → alinhamento → comentários e boas práticas.

# Melhorias e Correções – Slides Flexbox

## Sobre o documento

O arquivo **slides-flexbox-corrigido.md** contém 16 tópicos sobre ativação do Flexbox (display: flex), flex-direction, justify-content, align-items, gap, flex-wrap, flex-grow/flex-shrink/flex-basis, align-self, responsividade, Mobile First, breakpoints e media queries, align-content, order, navbar com Flexbox, Flexbox vs Grid e check-list de propriedades.

---

## Correções aplicadas

### Título de slide incorreto
| Problema | Correção |
|----------|----------|
| Título "Criação de slides: formatação, imagens, animações e transições" em bloco cujo conteúdo era sobre **barra de navegação com Flexbox** | Alterado para **"Aplicação: Barra de Navegação com Flexbox"** |

| Problema | Correção |
|----------|----------|

| Problema | Correção |

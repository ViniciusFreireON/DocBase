# Melhorias e Correções – Slides Estrutura de Projeto Web

## Sobre o documento

O arquivo **slides-estrutura-projeto-corrigido.md** contém 16 tópicos sobre organização de pastas (assets, css, js, img), tags semânticas (header, nav, main, article, section, aside, footer, figure, figcaption, blockquote, cite, q, time, details, summary), comentários, iframe (YouTube e Google Maps), links internos e âncoras.

---

## Correções aplicadas

### Sintaxe de comentários HTML
| Local | Erro | Correção |
|-------|------|----------|
| Slide sobre comentários | "delimitados por ``" (backticks vazios ou incompletos) | **delimitados por `<!--` e `-->`** (sintaxe correta em HTML) |

### Quebras de linha no meio de frases
Trechos unificados para leitura contínua:
- "publicá-lo"
- "Nossos Produtos e Contato"
- "Endereço de E-mail:, o cursor"

### Organização
- Conteúdo sobre tags semânticas de estrutura, figuras/citações e tempo/detalhes condensado em tópicos numerados.
- Sequência lógica: estrutura de pastas → semântica → mídia → links → âncoras → prática → check-list.

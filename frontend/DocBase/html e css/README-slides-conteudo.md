# Melhorias e Correções – Slides Conteúdo (HTML/Web)

## Sobre o documento

O arquivo **slides-conteudo-corrigido.md** contém 16 tópicos sobre introdução à Web, HTML, Front-End, Back-End, anatomia do documento e tags básicas (h1–h6, p, a, img).

---

## Correções aplicadas

### Erros de digitação
| Local | Erro | Correção |
|-------|------|----------|
| Slide "Criando Links (A tag \<a\>)" | `httpsa://www.google.com` | **https://www.google.com** (remoção do "a" em "https") |
| Slide "Hierarquia de Títulos" | `hierar_quia` | **hierarquia** (remoção do underscore indevido) |

### Ajustes de consistência e clareza
- **Formatação de código:** Tags HTML que apareciam quebradas entre linhas foram reorganizadas para leitura contínua (ex.: "Dentro de cada \<h2\>").
- **Formulário de login:** Os atributos `action="/processar-login.php"` e `method="POST"` mantidos corretos no texto dos slides.

---

## Melhorias no material

- Texto condensado em tópicos objetivos em Markdown.
- Sequência lógica preservada: Web → HTML → Front-End → Back-End → anatomia → tags básicas.
- Exemplos de código ajustados para evitar quebras desnecessárias.
- Nenhuma alteração conceitual; apenas correções pontuais e formatação.

---

## Conteúdo técnico já correto (sem alteração)

- Uso de `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`, `<meta charset="UTF-8">`, `<title>`.
- Descrição de cliente/servidor, request/response, HTML/CSS/JS no front-end e papel do back-end.
- Anatomia do documento, hierarquia de títulos, tag `<p>`, tag `<a>` com `href`, tag `<img>` com `src` e `alt`, comentários `<!-- -->`.

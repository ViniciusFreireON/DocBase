## Resumo das correções

- **Erros de digitação:** URL (httpsa → https), hierarquia (hierar_quia → hierarquia).
- **Quebras de linha:** frases e identificadores unificados para leitura contínua.
- **Organização:** parágrafos deslocados corrigidos, tabelas em Markdown.
- **Conteúdo:** títulos incorretos ajustados, exemplos adequados aos temas.
- **Boas práticas:** `lang="pt-br"`, comentários HTML (`<!-- -->`), especificidade e combinadores.

Para o detalhamento completo, consulte o [RELATORIO.md](RELATORIO.md) e os READMEs por documento.

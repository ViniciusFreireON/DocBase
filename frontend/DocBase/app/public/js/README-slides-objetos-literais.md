3. Acesso simples a propriedades (notação de ponto)  
4. Acesso a propriedades com colchetes [ ]

6. Modificando valores de propriedades existentes  
7. Removendo propriedades com o operador delete  
8. Métodos simples: adicionando comportamento  
9. Objetos dentro de arrays (lista de itens)  
10. Acessando dados em objetos aninhados  
11. Arrays dentro de objetos (propriedades-lista)

14. Caso de uso 2: modelando um produto de e-commerce

## Resumo das correções

| Tipo | Original | Correção |
|------|----------|----------|
| Título | O que são **O**bjetos **L**iterais | O que são **o**bjetos **l**iterais |
| Redação | "o string" | "a string" (concordância) |
| Typo | "vírgula (**.**)" | "vírgula (**,**)" |
| Quebra de linha | "{ nome: 'Mouse' , preco: 50 }" e " 'Teclado' , preco: 150" partidos | Texto contínuo **{ nome: 'Mouse', preco: 50 }** |
| Numeração | **1.**pedidos[0], **2..**itens, **3.**[0], **4..**produto, **5..**nome | **1.** pedidos[0], **2.** .itens, **3.** [0], **4.** .produto, **5.** .nome (espaço após o número) |
| Estrutura | Parágrafo sobre modelagem de usuário + "Caso de Uso 1..." separados | Unificado em um único slide (13) |
